Map

Map key sifatida hamma data typeni qabul qiladi

Unda Garbage collection ishlamaydi.

Map metodlari

  • new Map() – map yaratadi.
  • map.set(key, value) – mapga key va value qo'shadi.
  • map.get(key) – key orqali valueni chiqaradi agar mavjud bo'lmagan key kiritilsa undefined qaytaradi
  • map.has(key) – berilgan key map ichida bor yo'qligiga qarab true va false qaytaradi
  • map.delete(key) – key kritilsa map ichidagi property ni o'chirib yuboradi
  • map.clear() - mapni tozalaydi
  • map.size - mapni lengthini chiqarib beradi
WeakMap

WeakMap

Uning keyi har doim object bo'lishi shart va unda garbage collection ishlaydi

WeakMap metodlari

  • weakMap.set(key, value) - unga key va value qo'shadi
  • weakMap.get(key) - value ni oladi
  • weakMap.delete(key) - weakMapni tozalaydi
  • weakMap.has(key) - keyning bor yo'qligiga qarab true || false qaytaradi
Set

Set unique qiymat olish uchun ishlatiladi va faqat valuelardan iborat bo'ladi unda ham garbage collection ishlamaydi chunki unda key yo'q

  • new Set([iterable]) - set yaratish
  • set.add(value) - value qo'shadi
  • set.delete(value) - valueni olib tashlaydi va agar o'sha paytda value kiritilsa true aks holda false qaytaradi
  • set.has(value) - value ning bor yoki yo'qligiga qarab true||false qaytaradi
  • set.clear() - setni tozalaydi
  • set.size - setning lengthini aniqlab beradi
WeakSet

WeakSet xuddi setga o'xshaydi faqat unga value uchun object beriladi

Property Flags

Property Flags 3 ga bo'linadi

1. writable

Objectni qaysidir propertysini writable i true bo'lsa o'zgartirib(overwrite) bo'ladi false bo'lsa o'zgartirib bo'lmaydi

2. enumerable

Objectni qaysidir propertysini enumerable ini true qilinsa bu property loopda ko'rinadi agar false qiladigan bo'lsak u loopda ko'rinmaydi

3. configurable

Objectni qaysidir propertysini configurable true bo'lsa delete qilib bo'ladi agar false qilinsa o'sha propertyni delete qilib bo'ladi

Agar bitta propertyni o'zgartirmoqchi bo'lsak Object.defineProperty(object name, 'property key', {property flag: true or false }) dan foydalaniladi

Agar ko'proq propertyni o'zgartirmoqchi bo'lsak Object.defineProperties( object name, {
"property key:" { property flag: true or false } })
